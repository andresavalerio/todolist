# To Do List

The repository for the To Do List API, based on the Federal University of São Paulo (UNIFESP) course of OOP, maintained by Andresa Valerio, built with Python.

## About the App
This project is complementary to the [OOP repository](https://github.com/andresavalerio/oop).

## Use Cases
- [] sign up
- [] sign in
- [] create list
- [] create todo list
- [] create item in todo list
- [] show todo list
- [] change item description
- [] change item priority
- [] remove item from todo list
- [] mark item as done
- [] remove done mark in item
- [] remove todo list
